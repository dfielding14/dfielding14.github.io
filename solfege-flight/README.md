# Solfege Flight

Standalone static mini-app that lives alongside the main Jekyll site.

## Files

- `index.html` - shell and metadata
- `style.css` - app-specific styles
- `game.js` - gameplay, audio analysis, and UI logic
- `assets/` - social preview and icon assets

## Notes

- This app is intentionally isolated from the main site styles and layouts.
- If you add new assets, keep them inside `solfege-flight/assets/`.
